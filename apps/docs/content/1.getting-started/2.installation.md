## Quick Start

Add :name-text to your project using your preferred package manager:

::code-group
```bash [pnpm]
pnpm add notform
```
```bash [yarn]
yarn add notform
```
```bash [npm]
npm install notform
```
```bash [bun]
bun add notform
```
::

## Install a Validation Library

:name-text works with any standard-schema compatible validation library. Choose one based on your preferences:

### Zod (Recommended)

TypeScript-first schema validation with excellent type inference:

::code-group
```bash [pnpm]
pnpm add zod
```



## Framework Compatibility

NotForm is built for Vue 3 and works seamlessly with:

- [**Vue 3**](https://vuejs.org/) - Full support with Composition API
- [**Nuxt 3/4**](https://nuxt.com/) - Works out of the box (dedicated module with auto-imports coming soon)
- [**Vite**](https://vite.dev/) - Optimized for Vite-based projects
- [**TypeScript**](https://www.typescriptlang.org/) - Full type safety with zero configuration

## TypeScript Support

:name-text is written in TypeScript and provides full type safety out of the box. Your form state will be automatically typed based on your validation schema:

```ts
import { useForm } from 'notform'
import { z } from 'zod'

const schema = z.object({
  name: z.string(),
  age: z.number(),
  email: z.string().email()
})

const { state } = useForm({ schema })

// state is automatically typed as:
// {
//   name: string
//   age: number
//   email: string
// }
```

::tip
Enable `strict` mode in your `tsconfig.json` for the best type safety and developer experience.
::
